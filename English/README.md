#### 疑问句

##### 一般疑问句

- 谓语动词是Be动词时:**Am/Is/Are + 主语 + 其他 ? **
- 谓语动词是实义动词时: **助动词Do/Does + 主语 + 其他 ?**



##### 特殊疑问句(5w1h)

- 谓语动词是Be动词时: **特殊疑问词 + am/is/are + 主语 + 其他 ?**
- 谓语动词是实义动词时: **特殊疑问词 + 助动词do/does + 主语 + 动词原型 + 其他 ?**
